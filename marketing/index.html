Integration

One Line. That's All It Takes.

Drop-in replacement for any OpenAI-compatible client. Zero code changes beyond the URL.

integration.py 
# Before (paying full price for everything)
client = OpenAI(
    api_key="sk-your-key",
    base_url="https://api.openai.com/v1"
)

# After (73% savings, same quality)
client = OpenAI(
    api_key="your-pharos-key",
    base_url="http://localhost:3777/v1"  ← That's it
)
integration.js 
// Before (paying full price for everything)
const openai = new OpenAI({
    apiKey: "sk-your-key",
    baseURL: "https://api.openai.com/v1"
});

// After (73% savings, same quality)
const openai = new OpenAI({
    apiKey: "your-pharos-key",
    baseURL: "http://localhost:3777/v1"  ← That's it
});
terminal 
# Just change the URL — Pharos handles the rest
curl http://localhost:3777/v1/chat/completions \
  -H "Authorization: Bearer your-pharos-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "pharos-auto",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
